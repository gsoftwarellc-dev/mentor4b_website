# Dala Static Site

This folder is the publish root for the static site.

Run locally:

```sh
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/
```

For hosting, publish this `dala.craftedbygc.com` folder as the site root so absolute paths like `/images`, `/css`, `/scripts`, and `/models` resolve correctly.
